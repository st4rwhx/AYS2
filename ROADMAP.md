# AYS2 Development Roadmap

This roadmap outlines planned features, improvements, and bug fixes for AYS2. Items are prioritized by impact and feasibility.

**Last Updated:** July 2026  
**Vision:** Make PS2 emulation accessible and performant on iOS

## Current Status: 0.1.0 (Stable)

✅ MVP complete with core emulation working  
✅ SideStore integration live  
✅ Community infrastructure established

---

## 🎯 Planned Features

### Phase 1: Core Stability (Q3 2026)

- [ ] **Compatibility Improvements**
  - Expand game compatibility database
  - Improve graphics rendering accuracy
  - Fix audio sync issues

- [ ] **Performance Optimization**
  - JIT recompilation enhancements
  - Memory management improvements
  - Battery life optimization

- [ ] **User Experience**
  - In-app game library management
  - Save state management UI
  - Controller configuration presets

### Phase 2: Advanced Features (Q4 2026)

- [ ] **Netplay / Online Support**
  - Network adapter emulation
  - Online multiplayer testing

- [ ] **Recording & Streaming**
  - Screen recording support
  - Gameplay replay functionality

- [ ] **Developer Tools**
  - Debug console
  - Logging framework
  - Profiling tools

### Phase 3: Long Term (2027+)

- [ ] **Cross-Platform**
  - macOS (Mac Catalyst) support
  - Android version (ARMSX2 collaboration)

- [ ] **AI-Powered Features**
  - Game identification from ISO/CHD
  - Performance prediction
  - Automatic compatibility detection

---

## 🐛 Known Issues

### High Priority
- [ ] Some graphically intensive games struggle on base iPhone 11
- [ ] Audio crackling in certain titles
- [ ] Memory pressure on iPad Gen 7

### Medium Priority
- [ ] Controller mapping UI needs refinement
- [ ] Save state compatibility across versions
- [ ] Occasional game crashes on launch

### Low Priority
- [ ] Minor UI glitches in settings
- [ ] Tooltip inconsistencies
- [ ] Dark mode edge cases
