## Protocol Details

### RFC 3984 H.264/RTP Compliance

- Payload type: 97
- Clock rate: 90 kHz
- STAP-A format: Multiple NAL units in one packet
- FU-A format: Large NAL units fragmented
- Frame detection: IDR (type 5) = keyframe

### RTP Header (12 bytes minimum)

```

+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+

+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
```
